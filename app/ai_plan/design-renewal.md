# デザイン刷新計画 (Modern Corporate Style)

この計画では、Honoで作られた技術リスト管理アプリのデザインを「Modern Corporate」テイストに刷新し、Playwrightでその結果を確認します。

## 1. 目的
- 無機質な表を、視認性が高くモダンなビジネス向けUIに改善する。
- Playwrightを使用して、実装後のUIを自動で確認・記録する。

## 2. 実装方針
- **技術:** Vanilla CSS
- **デザイン:** 
  - コーポレートブルー (#2563eb) を基調とした配色。
  - 清潔感のある白背景と、階層構造を表現するための微妙なグレーの使い分け。
  - プロジェクト一覧（テーブル）の可読性向上（行のホバー、パディング、ボーダーの最適化）。
  - レスポンシブ対応（モバイルでも最低限見れるように）。

## 3. 実装ステップ

### ステップ 1: CSS の作成と配信設定
1. `src/style.css` を作成し、変数を定義して全体的なスタイルを記述する。
2. `src/index.tsx` にて `hono/serve-static` をインポートし、`/static` パスでCSSを配信できるようにする。

### ステップ 2: レイアウトとコンポーネントの刷新
1. `src/layout.tsx` を修正し、`<link rel="stylesheet" href="/static/style.css">` を追加。ナビゲーションバーを実装。
2. `src/components/list.tsx` を修正。テーブルにクラスを付与し、ボタン類をスタイリング。
3. `src/components/add.tsx` / `src/components/edit.tsx` を修正。フォーム入力をモダンなデザイン（ラベル配置、フォーカス状態など）に変更。
4. `src/index.tsx` 内のログインフォームも同様に刷新。

### ステップ 3: Playwright による検証
1. `tests/visual.spec.ts` を作成。
2. アプリを起動し、トップ画面（一覧）、追加画面、編集画面のスクリーンショットを撮影・保存するテストコードを記述。
3. テストを実行し、成果物を確認する。

## 4. 期待される成果
- プロフェッショナルで信頼感のあるUI。
- データの視認性が向上した一覧画面。
- 自動テストによる品質担保。
